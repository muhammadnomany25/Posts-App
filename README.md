
# STC Assignment (Posts App)

**This task helps to fetch some news data from the cloud and store them into local DB to support Online/Offline loading:**

### Architecture
* [MVVM](https://www.raywenderlich.com/8984-mvvm-on-android)

### Patterns
* [Repository](https://developer.android.com/jetpack/docs/guide)

### Approaches
* [SOLID Principle](https://itnext.io/solid-principles-explanation-and-examples-715b975dcad4?gi=79443348411d)
* [Clean Architecture](https://medium.com/android-dev-hacks/detailed-guide-on-android-clean-architecture-9eab262a9011)

* [Kotlin](https://kotlinlang.org/)
* [News API](https://newsapi.org)

* [Retrofit](https://square.github.io/retrofit/)
